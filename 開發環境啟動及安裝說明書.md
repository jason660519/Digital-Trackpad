# 萬能PC遙控器 - 開發環境啟動及安裝說明書

## 系統需求

- Python 3.8 或以上版本
- Windows 10 作業系統
- 網路連接（用於手機控制）

## 開發環境設定

### 1. 安裝 UV（Python 套件管理器）

UV 是一個快速的 Python 套件管理器，用於管理依賴和虛擬環境。

```bash
# 使用 pip 安裝 UV
pip install uv

# 或者使用官方安裝腳本（Windows）
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### 2. 複製專案

```bash
git clone https://github.com/rcsn01/Digital-Trackpad.git
cd Digital-Trackpad
```

### 3. 建立虛擬環境並安裝依賴

```bash
# 使用 UV 建立虛擬環境（會自動建立 .venv 目錄）
uv venv

# 啟動虛擬環境
uv shell

# 安裝專案依賴
uv sync
```

### 4. 啟動應用程式

```bash
# 使用 UV 執行應用程式
uv run python app.py --port 5001

# 或者直接啟動虛擬環境後執行
uv shell
python app.py --port 5001
```

## 應用程式使用說明

### 啟動後資訊

當伺服器成功啟動後，你會看到類似以下的訊息：

```
Starting trackpad server...
Access from your phone at: http://172.20.176.1:5001
Or access locally at: http://localhost:5001
Screen size detected: 1920x1080
 * Restarting with stat
 * Debugger is active!
 * Debugger PIN: xxx-xxx-xxx
```

### 連接方式

1. **本地測試**：在瀏覽器開啟 `http://localhost:5001`
2. **手機控制**：在手機瀏覽器開啟 `http://172.20.176.1:5001`（IP 位址會因網路而異）

### 功能特色

- **觸控板模式**：模擬筆電觸控板，支援多指手勢
- **簡報控制器**：PowerPoint/簡報控制
- **應用程式切換**：快速切換視窗
- **媒體控制**：音量、播放控制
- **自定義按鈕**：可自訂功能按鈕

## 常見問題

### Q: UV 安裝失敗？
A: 確保 Python 已正確安裝，並嘗試使用系統管理員權限執行安裝命令。

### Q: 依賴安裝失敗？
A: 檢查網路連線，或嘗試更新 UV：
```bash
uv self update
```

### Q: 手機無法連線？
A: 確保手機和電腦在同一個 Wi-Fi 網路下，並檢查 Windows 防火牆設定。

### Q: 觸控功能不正常？
A: 確保以系統管理員權限執行應用程式，某些功能需要較高權限。

## 注意事項

1. **安全性**：此應用程式會控制你的電腦，請確保在安全的網路環境下使用
2. **權限**：某些功能可能需要系統管理員權限
3. **防火牆**：首次執行時，Windows 防火牆可能會提示，請允許存取
4. **防毒軟體**：某些防毒軟體可能會誤報，請將此應用程式加入白名單
