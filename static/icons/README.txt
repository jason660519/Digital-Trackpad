Place app icons here. Recommended sizes:
- apple-touch-icon-180.png (180x180) for iOS Home Screen
- icon-192.png (192x192) for Android/Chrome
- icon-512.png (512x512) for Android/Chrome

If you don't have icons yet, you can create them from a source SVG or PNG using tools like ImageMagick or online generators. Example ImageMagick command:

convert source.png -resize 180x180 icons/apple-touch-icon-180.png
convert source.png -resize 192x192 icons/icon-192.png
convert source.png -resize 512x512 icons/icon-512.png
